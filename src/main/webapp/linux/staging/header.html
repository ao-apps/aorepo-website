AORepo Staging

The repositories within this staging channel contain the sets of packages being built-up during the weekly update packaging phase. The packages may have broken dependencies as we build new versions of packages and rebuild dependencies.

This channel is only used by our template and build environments.

Once all updates are packaged, the completed, coherent set of packages is promoted to the testing channel.

This channel is not intended for general use. Please use the production channel, and optionally the testing channel.
